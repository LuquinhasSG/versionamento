comandos descrição
git init cria um novo repositório Git
git clone cria uma cópia local de um repositório
git add adiciona arquivos ao índice
git commit cria um commit com os arquivos do índice
git status lista os arquivos que foram alterados junto com os arquivos que estão para serem commitados
git push envia os commits para o repositório remoto
git pull atualiza o repositório local com base no repositório remoto
git branch lista as branchs existentes no repositório
git checkout alterna entre as branchs existentes no repositório
git merge une duas branchs
